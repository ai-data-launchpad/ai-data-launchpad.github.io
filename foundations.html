Hands-on Labs: AI & Data Technical Foundations

Deep-dive into Salesforce Einstein, Agentforce & Data Cloud

About the Salesforce AI & Data Technical Foundations Labs

Alongside the core concepts, attendees will dive deep into the underlying AI and data technologies.

This program goes beyond configuring topics and actions — it builds a deep understanding of how autonomous agents operate, including how they select topics, choose actions, and leverage Retrieval-Augmented Generation (RAG) for augmenting LLM capabilities. You’ll leave ready to design and launch autonomous agent experiences for your customers and employees.

In-depth topics covered include the Data Cloud data model, Vector Databases, Data Cloud Sandboxes, Data Cloud One, Prompt Templates, the Atlas Reasoning Engine, the Agentforce Testing Center, agent anatomy (best practices for custom Topics and Actions), Retrieval-Augmented Generation (RAG) (ingestion options, semantic search indexes, retrievers), and the Agentforce Data Library.

Hands-on Labs Overview (Technical Foundations)

  • Data Cloud: Data Ingestion with CRM Connector, Data Harmonization, Profile Unification & Org Enrichment for customer 360.
  • Agentforce: Custom Topics/Actions, Einstein Trust Layer, Einstein Feedback, Agentforce Conversation Explorer, Prompt Templates, Agentforce Data Library, Agent Deployment (Experience Cloud).

    The labs will cover Agentforce use cases for both - internal users (employees) and external users (customers).

    We shall also demo/discuss additional agents such as Agentforce SDR Agent, Sales Coach Agent, Service Planner (Agentforce Service Assistant) - however hands-on labs are not planned for these agents.

Agent Maturity Stages

Hands-on labs allow participants to explore key stages of agent maturity — from configuring agents to assist users using a company's knowledge base (using Data Library), to enabling autonomous action through Salesforce Flows and Apex.

The foundational labs prepare participants for more advanced topics, such as headless agents and multi-agent systems, which will be covered in subsequent sessions.
