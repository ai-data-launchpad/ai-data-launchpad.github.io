Hands-on Labs: AI & Data Technical Foundations

Deep-dive into Salesforce Einstein, Agentforce & Data Cloud

About the Salesforce AI & Data Technical Foundations Labs

In addition to fundamental concepts, attendees dive deep into the AI & data technologies.

Some of the in-depth topics include Data Cloud data model, Vector Database, Data Cloud Sandboxes, Data Cloud One, Agentforce Reasoning Engine, Agentforce Testing Center, agent anatomy (best practices for Prompt Templates, Topic instructions, Agentforce Actions), RAG & Data Library.

The labs will allow participants to explore Agent maturity stages e.g., configuring agents to "assist" using company's knowledgebase, and autonomously "take action" with Flows and Apex.

Lab participants will prepare for the advanced topics such as headless agents and multi-agent systems to be covered in the next lab.

Hands-on Labs Overview (Technical Foundations)

  • Part -I Data Cloud: Data Ingestion with CRM Connector, Data Harmonization, Profile Unification & Org Enrichment for customer 360.
  • Part -II Agentforce: Custom Topics/Actions, Prompt Templates, Agentforce Data Library, Agent Deployment (Experience Cloud). The labs will cover Agentforce cases for both internal and external users i.e., Default Agent for Employees and Agentforce Service Agent for Customers. We shall also demo/discuss additional agents such as Agentforce SDR Agent, Sales Coach Agent, Service Planner (Agentforce Service Assistant) - however hands-on labs are not planned for these agents.
